Rangsk's Puzzle Solver

This website is a fork of the puzz.link puzzle editor and solver UI, without the database.

Purpose

The intended purpose of this website is to make constructing these types of pencil puzzles easier by use of an integrated solver, written in Rust and compiled to webassembly. No external software is required!

Compatibility

All puzzles created here are compatible with puzz.link, and so can be used in their database and solving interface once constructed using this website.

Solver Support

This page is a work in progress, and currently only supports the solver for the following puzzles:
  • None (coming soon!)

How To Use

Visit the creation tool to start creating a puzzle of whichever type you'd like.
